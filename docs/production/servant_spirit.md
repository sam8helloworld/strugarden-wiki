# 使役精霊

生産スキルで使用する精霊のページ。レシピごとに特定の使役精霊が必要となり、精霊との親密度が生産に影響する。

## 使役精霊各種

- **サラマンダー**：炎の精霊。物質を変化させる力を持つ
- **ウィンディーネ**：水の精霊。異なる物質同士を溶け合わせる力を持つ
- **ノーム**：地の精霊。異なる物質同士を共存させる力を持つ
- **シルフ**：風の精霊。物質の形状を変化させる力を持つ
- **プフリ**：雪の精霊。物質の時間を止める力を持つ
- **ドリアード**：木の精霊。溶けあった物質を分ける力を持つ
- **ウィルオーウィスプ**：光の精霊。物質の時間を巻き戻す力を持つ
- **シェイド**：闇の精霊。物質の時間をすすめる力を持つ

## 親密度

### 基本事項
- 親密度は小数点第一位まで存在するが画面に表示されない
- 親密度上昇チャンス：完成時および新規作成失敗時
- **親密度上限** = 製作可能レベル ÷ 2 + 3.5
  - 装備品の場合：装備レベル ÷ 2 + 1
- 親密度が高いと失敗時のロスト量が減少

### ファミリア合成サポート

各ファミリアは特定の使役精霊の親密度上昇をサポートする：

| 精霊 | 対応ファミリア |
|------|-------------|
| サラマンダー | 鳥 |
| ウィンディーネ | 鳥、ハーピー、うさぎ |
| ノーム | 鳥、ハーピー、うさぎ、くま |
| シルフ | くま、たぬき |
| プフリ | くま、たぬき |
| ドリアード | ゴブリン |
| ウィルオーウィスプ | アルマジロ |
| シェイド | 全種対応可能 |

## 親密度に関する情報

### S装備の確率
- 親密度0でもS装備作成は可能
- 基本的には運の要素が大きい
- レベルが高いほどS装備が出やすい傾向

### 親密度上昇速度
- 上昇上限マイナス4〜マイナス2の適正範囲では高速上昇
- マイナス1.9以上でレシピ成功率が向上する傾向

### その他
- 特定のレベル帯で上昇が加速することがある
- 同じレシピを繰り返すと平均80回前後で親密度が上昇する傾向
- 高レベル装備でも親密度0から作成可能だが、初期段階での失敗率は高い
