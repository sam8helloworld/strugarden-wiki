# 錬金術

合成スキル「錬金術」のページ。ポーションの調合、銃・アクセサリの生産が可能。調理や彫金などの他の生産職と異なり、素材やレシピの入手方法が特殊な物が多いため低レベルから始めるのは困難。

## 錬金術秘伝書の入手法

### 購入
- 200,000z（ウベン中央道具屋）

### ドロップ
- ドライボーン（パプティス砂丘）
- モーシィボーン（スタンクライド古戦場）
- スケルトンソルジャー（ファストベル迷宮）
- ポーンスケルトン（ロ・ナデ地下要塞）
- ハウスボーン（メイリンの館）

## 情報提供

レベル上昇範囲についての報告方法：
- **確定情報**: Lv1.9で「難しすぎるので作製を中止」、Lv2.0で作製可能な場合は「2.0〜10.0」と記述
- **不確定情報**: 確認できた最高値に「×」、最低値に「○」を付与（例：×3.8、○4.0）
- スキル上昇値が判明している場合は「2.0↑0.3」と記述
