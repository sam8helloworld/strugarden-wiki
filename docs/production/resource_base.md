# 資源拠点

各地域に設けられた生産素材の採取専用エリア。

## 拠点への移動

自国が支配する拠点には資源拠点案内係に話しかけることで移動できる。

### 移動条件

- 移動したい拠点へ行ったことがある
- 自国が支配している拠点である
- 資源拠点への移動料金を支払う
- ※他国首都の資源拠点案内係に話しかけても自国の資源拠点に移動可能
- ※資源拠点入場許可証がある場合、無料で入ることが可能
