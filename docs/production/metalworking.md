# 彫金鍛冶

合成スキル「彫金鍛冶」のページ。鉱石からインゴットを精錬・鍛錬し、武器・防具・アクセサリを生産する。

## レシピ

### 基礎概念

- **精錬**：鉱石からインゴットを生成するプロセス
- **鍛錬**：インゴットをより高品質に仕上げるプロセス
- スキルレベルが高いほど高品質な産出率が向上
- TD使用で粗ダマ精錬から高純度ダマ鍛錬まで段階的にレベルアップ可能
- スキル上昇は「作成レベル+10」が標準

### Lv1〜10 製品

ダブルアックス、ナイフナックル、ソルジャーヘルム、ロングソード、グレートアックス、スレイヤー、ククリ、大刀、ネーザルヘルムなど。銅・青銅を主材料とする。

### Lv11〜20 製品

ソードナックル、ブロードソード、ツイン・トマホーク、ヒュージアックス、ナランジャソードなど。黄銅・高硬度青銅を多用し始める。

### Lv21〜30 製品

バットカタール、ブラストアックス、クレイモア、輪状鎧シリーズ、赤刃など。銀・高純度鉄を活用した装備が増える。

### Lv31〜40 製品

タバール、バトルソード、ナイツヘルム、ナイトプレート、黒刃など。金・白金の使用が本格化。

### Lv41〜50 製品

グラディウス、蒼龍シリーズ、レイニーブルー、地裂の斧など。白金・ミスリルが主要材料。

### Lv51〜60 製品

蝶遷斧、グロウメイル、バーサーカーブレード、ドラゴンヘルム、スカーレットニードル、村正、天上天下、無双刃など。ダマスカス鋼・高硬度ミスリルを活用した高度な装備。
