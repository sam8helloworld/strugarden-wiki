# デッキ

## デッキとは

デッキとは戦闘のために用意したスキルの組み合わせのこと。戦闘に持ち込めるスキルの枚数と種類は、デッキシートの種類によって異なる。

- 最大5枚まで習得できる
- 町の出口付近にいる「デッキ・○○○」NPCに話しかけることで、シートの選択とスキルの組み換えが可能
- 誤って削除したデッキは復元できない
- メインクエストの報酬として入手できるデッキは、ジョブを問わず1回のみ入手可能
