# 誕生石

## 概要

キャラクターの誕生日記念に、誕生石をプレゼントと交換できるシステム。誕生日はキャラクター作成後の最初のログイン時に決定し、ステータス画面で確認できる。

## 入手・交換手順

1. キャラクターの誕生月に対応した**誕生石の欠片**を20個集める
2. 各国のNPCに大きな誕生石と交換してもらう
3. 誕生日に「幾度目かの誕生日」称号を取得
4. 誕生日から40日以内（リアル72時間）に、各国のNPCに誕生石を渡してアイテム交換

- アイテム交換は年に1回のみ
- 交換可能期間を過ぎると称号を失う
