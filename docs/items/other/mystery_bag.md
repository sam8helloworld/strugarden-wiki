# 謎の袋

## 概要

謎の袋は、酒場クエストの報酬権利書を選択メニューで変換することで入手できる。元の権利書の星の数にかかわらず、最初に受け取る袋は常に「「い」の一」。

袋を開けると確率でさまざまなアイテムが入手でき、さらに次の番号の袋も含まれる場合がある。「くず鉄」（NPC売値30 ZULA）はハズレアイテム。

## 素材アイテム詳細

### ルーン糸
- スタック上限：100個 / NPC売値：55〜135 ZULA
- 紡績裁縫・皮革加工の素材として使用
- アイリス〜ソル（Lv3〜Lv56対応アイテム向け）

### 聖水
- スタック上限：100個 / NPC売値：100〜600 ZULA
- 神父の〜女神の グレード
- 紡績裁縫のローブ（Lv4〜Lv56）や錬金術アイテムに使用

### 戦魂
- スタック上限：100個 / NPC売値：50 ZULA
- 鈴〜ダスト グレード
- 紡績裁縫の武道着（Lv14〜Lv60）に使用
