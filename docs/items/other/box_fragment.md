# 函の欠片

## 概要

ナテルア神が人間の王に贈ったとされる「アニマ・クラテラ（通称：函）」が冒険者アグナ・フィーチャーによって粉々に砕かれ、世界中に散らされた。ラデニウス示教国の教会が欠片の収集を支援しており、収集数が多い者には助成金が支給される。

## 函の欠片の種類

| 名称 | 説明 | 備考 |
|------|------|------|
| SSパーツ | アニマ・クラテラの函の欠片（サイズ極小） | 重さ：1ベルデ、敵ドロップ（ダーティバニー・ダンディウサギ・カースガーゴイル） |
| Sパーツ | 砕け散った函の欠片。教会で預かってくれる。小さめ。 | 重さ：10ベルデ |
| Mパーツ | 砕け散った函の欠片。教会で預かってくれる。中ぶり。 | 重さ：30ベルデ |
| Lパーツ | アニマ・クラテラの函の欠片（サイズ大） | 重さ：50ベルデ |
| 黒い函の欠片 | 黒く染まってしまった函の欠片 | ウベンメインクエスト第3話入手。1ベルデ＝30パークポイントに換金可 |

## 取得方法

- 各国メインクエスト
- ミニクエスト報酬「がらくた」
- ギルドダンジョン
- 敵ドロップ（SSパーツのみ）
