# モンスターの知識

## 概要

モンスターを「ファミリアシーリングカード」に封印するには、対象モンスターの「モンスターの知識」アイテムが必要。

**使用手順**
1. アイテム欄で「モンスターの知識」をダブルクリックして同名の共通スキルを習得
2. デッキにそのスキルをセット
3. モンスターとの戦闘時にスキルを使用して封印

種族ごとに異なる知識が必要。別種族の封印には別の知識が必要。
