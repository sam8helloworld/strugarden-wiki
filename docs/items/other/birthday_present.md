# 誕生プレゼント

## 概要

誕生日から40日間（リアル72時間）、「幾度目かの誕生日」称号を得る。この期間中に各国の教会にいる「○○ミカド」NPCに[誕生石](./birthstone.md)を渡すと、ハッピーキャンドル・バースデーケーキ・装備品などのプレゼントを受け取れる。

→ NPCの場所は[誕生石](./birthstone.md)ページを参照。

## ハッピーキャンドル

- スタック上限：20個
- 現在の用途：不明

## バースデーケーキ

食べ物アイテム。詳細は食べ物ページを参照。
