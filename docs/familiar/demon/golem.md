# ゴーレム系ファミリア

## 概要

ゴーレム系の捕獲はできない。以下の方法で入手可能。

- **ゴーレムクエスト**：マインゴーレム・ニューマインゴーレムのシーリングカード入手
- **ラッキースクラッチＡ賞**：ゴールデンゴーレムのシーリングカード入手
