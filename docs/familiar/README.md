## ファミリアとは
--------------------------------------------------------------------------------------------------------------------

一緒に戦ってくれたり、生産サポートをしてくれたりするペットモンスターです。  
一部のモンスターは、ファミリアとして飼う事が出来ます。

*   ファミリアの使い道
    *   戦闘
        *   やられても次の戦闘で復活するため主人の盾となってくれます。
        *   回復スキルや攻撃スキルを覚えて助けてくれます。  
            ※ファミリアを召喚すると毎ターンにＡＰ３を消費します。
    *   生産
        *   合成サポートのスキルを覚えさせれば生産を手助けしてくれます。
        *   資源拠点で素材交換の看板としても使えます。
    *   コミュニケーション
        *   ファミリアの名前を変更すればパーティ募集やバザーの広告に使えます。

### ファミリア育成スキル

ファミリアはレベルアップ時のステータスでスキルを覚えます。  
育成経験値は「しぬほどあそぶ」「おつかいさせる」で獲得でき、１０００ポイントでレベルアップします。

*   自動召喚する
    *   ファミリアを最初から戦闘に参加させることが出来ます
*   合成サポート
    *   ファミリアが合成サポートスキルを覚えている場合、このコマンドを使ってサポートしてもらいます
*   おやすみ
    *   ファミリアの体力を少し回復させます
    *   24分（ストラ時間で8時間）経つと起きます
*   しぬほどあそぶ
    *   動けなくなるまで遊びます
    *   クール/やさしさの性向値が変化します
        *   すこし：１～１０
        *   そこそこ：１０～２０
        *   普通：２０～３０
        *   とても：３０～４０
        *   ものすごく：４０～５０
*   おつかいさせる
    *   お使いに出したエリアの拠点から何か一つ採取してきます。稀に、お使いでしか手に入らないアイテムを持ってきます。
    *   24分（ストラ時間で8時間）経つと帰ってきます
    *   ワイルド/かしこさの性向値が変化します
        *   すこし：１～５
        *   そこそこ：５～１０
        *   普通：１０～１５
        *   とても：１５～２０
        *   ものすごく：２０～２５
*   食べ物をあたえる
    *   疲れにくくなる効果がある？？
    *   親密度が上がるといわれていますが、よくわかりません
*   なまえをつける
    *   何度でも付け替えられます
*   ファミリアを置く
    *   キャラクターの正面にファミリアを置く事ができます
*   修得スキル表示
    *   覚えたスキルを見る事ができます
*   契約破棄
    *   ファミリアを逃がします。1度逃がすと2度と戻ってくる事はありません
*   連れて歩く
    *   プレミアム会員限定機能です。ファミリアが後ろからついてきます

### 種族別スキル

各種族のページにはスキル、初期ステータス、ＨＰなどの情報がまとめられています。

覚えるスキルは育成レベルとステータスによって決まっているため、  
低レベル帯限定スキルや高レベル帯限定スキルがほとんどです。  
スキルは覚えないこともありますので、スキルの習得レベルとステータスを確認しましょう。

*   飛行系
    *   鳥系
    *   ハーピー系
*   動物系
    *   うさぎ系
    *   たぬき系
    *   くま系
*   亜人系
    *   ゴブリン系
    *   アルマジロ系
*   両生系
    *   ヘビ系
    *   カエル系
    *   カメ系
*   妖精系
    *   プチドラゴン系
*   魔生系
    *   ゴーレム系
