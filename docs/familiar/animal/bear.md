# くま系ファミリア

## 特徴

斬・衝・電・冷属性に強く、毒・炎属性に弱い。

## 入手方法

現在、くま系は通常捕獲ができない。エンシェントパンダはパンダミニクエストのドロップからのみ入手可能。
