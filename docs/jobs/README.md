## 職業(ジョブ)について [†](https://wiki.strugarden.pluslake.net/%E8%81%B7%E6%A5%AD%23u7123c50 "u7123c50")

ジョブとは？  
プレイヤーキャラはスタート時から必ずいずれか一つの**ジョブ**に就くことになります。  
ジョブごとに修得できるスキルや装備[アイテム](https://wiki.strugarden.pluslake.net/%E3%82%A2%E3%82%A4%E3%83%86%E3%83%A0 "アイテム (1682d)")、最大HP、移動力などが違います。  
選択した**メインジョブ**は変更することはできないので慎重に選びましょう。

サイドジョブとは？  
一定の制限下で**サイドジョブ**へジョブチェンジを行うことができます。  
サイドジョブとは下のリング上で隣合っている職業のことで、各国首都にいるNPCに話し掛けることでジョブチェンジが行えます。  
（例：盗賊なら黒印と戦士）  
サイドジョブのレベルアップはメインジョブの半分までで、レベルアップに必要な経験値はメインジョブの1.5倍です。  
経験値はレベルアップ直前までためることができます。  
サイドジョブを一定までレベルアップさせないと、覚える事ができないスキル（**サイドスキル**）もあります。

上位職とは？  
LV45の状態で[ドラゴニアメインクエスト第1話](https://wiki.strugarden.pluslake.net/%E5%9B%BD/%E3%83%89%E3%83%A9%E3%82%B4%E3%83%8B%E3%82%A2/%E3%83%A1%E3%82%A4%E3%83%B3%E3%82%AF%E3%82%A8%E3%82%B9%E3%83%88%23v4ec022a "国/ドラゴニア/メインクエスト (996d)")をクリアすると、基本職から上級職へ転職できます。  
上級職専用のスキルも多数存在します。

## 各職業の特徴 [†](https://wiki.strugarden.pluslake.net/%E8%81%B7%E6%A5%AD%23e031032d "e031032d")

### [戦士](https://wiki.strugarden.pluslake.net/%E8%81%B7%E6%A5%AD/%E6%88%A6%E5%A3%AB "職業/戦士 (1624d)") [†](https://wiki.strugarden.pluslake.net/%E8%81%B7%E6%A5%AD%23k45c205e "k45c205e")


多彩な武器と強固な防具に身を包む近距離戦闘のエキスパート。  
重装備の戦士は、圧倒的な防御力を持ちますが、戦闘時の移動範囲が極端に少なく、  
単独で敵を追いつめるには経験と勘が必要となる。

### [格闘士](https://wiki.strugarden.pluslake.net/%E8%81%B7%E6%A5%AD/%E6%A0%BC%E9%97%98%E5%A3%AB "職業/格闘士 (1653d)") [†](https://wiki.strugarden.pluslake.net/%E8%81%B7%E6%A5%AD%23d7b9cfae "d7b9cfae")


攻撃の威力はほかの職業と比べてもダントツ。  
成長すると気功を放出して遠距離攻撃も可能になる。  
ただし防御力が弱く、戦場の最前線では苦戦する��能性大。

### [守護魔導師](https://wiki.strugarden.pluslake.net/%E8%81%B7%E6%A5%AD/%E5%AE%88%E8%AD%B7%E9%AD%94%E5%B0%8E%E5%B8%AB "職業/守護魔導師 (1268d)") [†](https://wiki.strugarden.pluslake.net/%E8%81%B7%E6%A5%AD%23n5896c2e "n5896c2e")


回復・治療・能力上昇・バリアを駆使し、仲間を影から支える。  
他の魔導師と違い、移動力は劣るものの、防御力の高い防具を若干装備できる。  
しかし、攻撃の種類が少ないため、単独で冒険に出るのは非常に危険。

### [精霊魔導師](https://wiki.strugarden.pluslake.net/%E8%81%B7%E6%A5%AD/%E7%B2%BE%E9%9C%8A%E9%AD%94%E5%B0%8E%E5%B8%AB "職業/精霊魔導師 (1776d)") [†](https://wiki.strugarden.pluslake.net/%E8%81%B7%E6%A5%AD%23q2d3a04b "q2d3a04b")


攻撃魔法は威力が高く、広範囲に効果をもたらすものもある。  
複数の敵を一度に倒せることが最大の利点だが、詠唱中に攻撃を受けると魔法を失敗してしまうことも。  
ただし防御力は全ジョブでも最低クラス。

### [黒印魔導師](https://wiki.strugarden.pluslake.net/%E8%81%B7%E6%A5%AD/%E9%BB%92%E5%8D%B0%E9%AD%94%E5%B0%8E%E5%B8%AB "職業/黒印魔導師 (1367d)") [†](https://wiki.strugarden.pluslake.net/%E8%81%B7%E6%A5%AD%23o85f1f2f "o85f1f2f")


闇の力を使って描く黒印（魔法陣）で敵を罠に陥れる。  
黒印の上に乗った敵に、ダメージや能力弱体化などのペナルティを与えることができるが、  
黒印は消滅するまで描いた者の力を吸収し続けるため、戦場での駆け引きが難しくなる。  
敵が確実に罠にかかるよう、行動を常に予測する必要がある。

### [盗賊](https://wiki.strugarden.pluslake.net/%E8%81%B7%E6%A5%AD/%E7%9B%97%E8%B3%8A "職業/盗賊 (1701d)") [†](https://wiki.strugarden.pluslake.net/%E8%81%B7%E6%A5%AD%23z8126b1e "z8126b1e")


遠距離攻撃と変則的な行動、俊足を備え持つ。  
移動量の多さをうまく利用して、敵との距離を保ちながら遠距離攻撃を仕掛けることができる。  
また、ほかの職業では修得できない特殊な行動も数多く存在する。
